# Competitor Alternatives

## When to Use

Use this skill when a user asks for competitor comparison, alternatives research, or positioning support.

## Workflow

1. Identify the primary product and target segment.
2. List top alternatives and summarize differentiators.
3. Produce recommendation points for outbound messaging.

## Output

- Competitor table (feature/pricing/target segment)
- Risks and opportunities
- Suggested positioning statements
